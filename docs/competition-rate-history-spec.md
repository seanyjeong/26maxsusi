# 수시 전년도 경쟁률 Spec

## Problem

수시 상담의 경쟁률 표기가 `25학년도`로 고정되어 2027학년도 화면에서도
오래된 값으로 보인다. 제공된 엑셀의 2024~2026 이력을 검증 가능한 형태로
보존하고, 선택 학년도의 직전 연도 자료만 학교 정보 화면에 표시해야 한다.

## Scope

- 2026 카탈로그는 2025, 2027 카탈로그는 2026 경쟁률을 사용한다.
- 상담 간단요강, 모집요강 탐색, 라이브의 선택 학교 정보에 같은 값을 표시한다.
- 엑셀에 없는 값이나 검토되지 않은 유사도 매핑은 만들지 않는다.

## Acceptance Criteria

1. 경쟁률은 `지원자수 / 모집인원`을 Excel과 같은 방식으로 소수 둘째 자리 반올림한다.
2. 2027 화면은 2026 자료만, 2026 화면은 2025 자료만 표시한다.
3. 기준 연도가 다르거나 데이터가 없으면 `전년도 경쟁률 자료 없음`으로 표시한다.
4. 상담·탐색·라이브가 같은 대학ID에 같은 모집·지원·경쟁률을 표시한다.
5. 기존 `/susi/university-details`, `/explore-universities` 응답 형태는 유지한다.
6. 라이브는 상세조회로 값을 보강하며 `_college_list`와 운영 백엔드는 변경하지 않는다.

## Migration And Rollback

- 원본 이력과 source hash를 보존하고, 변경 전 카탈로그 행을 별도 백업한다.
- 롤백은 파생 필드를 복원한 뒤 이번 source hash의 이력 행을 제거한다.
